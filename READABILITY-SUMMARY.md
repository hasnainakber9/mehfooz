## 🔧 TROUBLESHOOTING

**If text still hard to read:**
1. Increase contrast:
   ```css
   --text-primary: #FFFFFF; /* Pure white */
   ```

2. Add stronger shadows:
   ```css
   h1 { text-shadow: 0 2px 4px rgba(0, 0, 0, 0.9); }
   ```

3. Darken backgrounds:
   ```css
   .hero::before {
     background: rgba(0, 0, 0, 0.8); /* Darker overlay */
   }
   ```
